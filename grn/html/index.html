体育品牌

( 贵人鸟、And1、Price等 )

多品牌战略是全能体育深耕中国体育市场的重要战略之一：通过不同品牌覆盖不同的细分市场，把握大众及中高端体育用品市场的庞大机遇。未来公司将继续收购多品牌，提升市场占有率和公司盈利水平。
